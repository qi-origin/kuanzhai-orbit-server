# 六爻卦理

---

## 第一节 日建、月建与日破、月破

### 1. 日建与月建

- 月建：占卦当月的地支  
- 日建（或日辰）：占卦当天的干支  

它们是判断用神**旺衰的核心标准**。

例：

辰月甲寅日占得《地天泰》变《地泽临》：
子孙――酉（应）
妻财――亥
兄弟――丑
兄弟──辰（世）化丑土兄弟
官鬼──寅
妻财──子
复制
- 辰为月建  
- 寅为日建  

规律：

- 临月建或日建 → 一般为旺  
- 古人云：  
  - “月令乃卜筮之提纲”  
  - “日辰为六爻之主宰”

### 2. 月破与日破

- **月破**：与月建相冲的爻 → 极弱、不利  
- **日破**：与日建相冲的爻 → 状态多变，但总体不利

## 第二节 旬空（空亡）

### 1. 概念

六十甲子中，天干10、地支12 → 每旬会有两个地支“空缺”，称为：

- 旬空 / 空亡

判断方法：

- 以起卦日辰为基准
- 查旬空表

### 2. 旬空表（示例）
甲子旬：空戌亥
甲戌旬：空申酉
甲申旬：空午未
甲午旬：空辰巳
甲辰旬：空寅卯
甲寅旬：空子丑
复制
### 3. 真空与假空

#### 假空

满足以下任一：

- 旺 → 不为空
- 动 → 不为空
- 得日月生扶 → 不为空

#### 真空

- 月破 → 真空
- 有气但不动 → 空
- 伏而受克 → 空
- 季节真空：
  - 春土
  - 夏金
  - 秋木
  - 冬火

## 第三节 进神与退神

### 1. 进神

动爻变化为：

- 寅→卯
- 巳→午
- 申→酉
- 亥→子
- 丑→辰
- 辰→未
- 未→戌
- 戌→丑

👉 表示：发展、进步

### 2. 退神

反向变化：

👉 表示：退化、倒退

### 3. 判断原则

- 用神 / 原神 化进 → 吉  
- 用神 / 原神 化退 → 凶  
- 忌神 / 仇神 化进 → 凶  
- 忌神 / 仇神 化退 → 吉  

---

## 第四节 飞神与伏神

六爻中不一定六亲齐全。

若用神不在卦中：

👉 需从宫卦中“找出来”

- 找到的 → **伏神**
- 原位爻 → **飞神**

👉 本质：隐藏信息与显性信息的关系

---

## 第五节 三合局

三合：

- 亥卯未 → 木局
- 寅午戌 → 火局
- 巳酉丑 → 金局
- 申子辰 → 水局
- 辰戌丑未 → 土局

### 成局方式

1. 三动爻成局  
2. 两动爻成局  
3. 一动爻成局  
4. 上卦变动成局  
5. 下卦变动成局  

### 吉凶判断

- 合成用神 / 原神 → 吉  
- 合成忌神 / 仇神 → 凶

## 第六节 反吟与伏吟

### 一、反吟

#### 1. 卦反吟

特征：

- 主卦与变卦**六爻相冲**
- 属六冲结构

代表：

- 反复
- 不稳定
- 动荡

#### 2. 爻反吟

- 单爻相冲变化

---

### 二、伏吟

特征：

- 卦变后**地支五行不变**

可分：

- 全卦伏吟
- 上卦伏吟
- 下卦伏吟

代表：

- 压抑
- 忧虑
- 停滞

---

## 第七节 合处逢冲 / 冲处逢合

- 合 → 聚、和
- 冲 → 散、破

### 规律

- 合处逢冲 → 先好后坏  
- 冲处逢合 → 先坏后好  

例：

《天地否》测婚：

- 本为六合 → 看似好  
- 但世应受冲 → 结果失败
