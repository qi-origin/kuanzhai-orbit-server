### 5. **六十四卦表 (`HEXAGRAMS`)** — **核心缺失** ⚠️

> 知识库 [装卦方法.md](../../base_knowledge/装卦方法.md) 的"八宫卦序"段已列出**每宫八卦的名字**，但没有每卦的：
> - 上卦（Trigram）+ 下卦（Trigram）
> - 卦宫归属（palace / palaceType: 本宫/一世/二世/...）
> - 卦名 → 五行

需要一份 64 卦的硬表（数组 / map），每行结构：

```ts
type HexagramMeta = {
  id: number;                    // 1~64，传统编号
  name: string;                  // 中文卦名 (例 "雷水解")
  upper: Trigram;
  lower: Trigram;
  palace: '乾宫'|'坎宫'|...;     // 该卦所属八宫
  palaceType: '本宫'|'一世'|'二世'|'三世'|'四世'|'五世'|'游魂'|'归魂';
};
```

来源建议：
- 知识库没有现成全表
- 网络搜索："六十四卦 卦象 上下卦 世应" / "周易 64卦 卦宫 世爻位置"
- 经典参考：朱熹《周易本义》、郭雍《京氏易传》、刘伯温《黄金策》

**当前状态**：❌ 缺完整表 — 需要你提供或确认外部数据源

### 6. **六十四卦世爻、应爻位置表** — **核心缺失** ⚠️

> 知识库 [装卦方法.md](../../base_knowledge/装卦方法.md) 装世应段有口诀：
> "天人归一，天同二；天地在三，错在三；地同人同世在四；地人相同世在五。"
> + 八卦之首世六当，以下初爻轮上扬；游魂八卦四爻立；归魂八卦三爻详。

口诀能算，但有歧义（"天"和"人"指上卦的哪一爻、还是上卦三爻的整体？）— 建议：

**方案 A**（推荐）：直接用第 5 项里的 64 卦表 + palaceType 推算
- 本宫 → 世 6
- 一世 → 世 1
- 二世 → 世 2
- 三世 → 世 3
- 四世 → 世 4
- 五世 → 世 5
- 游魂 → 世 4
- 归魂 → 世 3
- 应爻 = 世爻+3 (mod 6, 1-indexed)

✅ **所以填完 #5 之后，#6 就自动免费**。建议先填 #5。
