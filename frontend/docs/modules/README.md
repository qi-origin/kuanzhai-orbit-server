# Module API Handoff Docs

This folder keeps backend handoff contracts for UI simulation modules.
Use `docs/README.md` for the top-level docs index.

## Rules

- Every UI behavior change must update its module doc.
- Code placeholders must use `TODO(Backend Integration)[module_api#anchor]`.
- Anchor keys are stable and should not be renamed without migration notes.
